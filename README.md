# JSOF
liberal JSON

## API
### value = jsof.parse(text)
The `jsof.parse()` method parses a JS string and returns a JavaScript value.

### text = jsof.stringify(value)
The `jsof.stringify()` method converts a JavaScript value to a JS string.
